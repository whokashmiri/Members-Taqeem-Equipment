If you have directly added Extension from Chrome Store, Install Host (Does not require JRE)

Download and Install Extension Host

Used for Digital Signature of PDF, eReturns, PKI Web Authentication, RSA Encryption/Decryption, Certificate Enrollment/Download form USB Token or Smartcard connected to users device from Web Browser through Web Application.

Using this extension on your own Website requires that Web Developers integrate JavaScript API provided by this Extension in your Web page.
